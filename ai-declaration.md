# AI Declaration
This file explicitly states the allowed usage of AI in this project. If you plan on contributing to this repository, please read carefully.

## AI Usage
The use of AI is permitted in this project in the following ways:
- Ideation
- Code breakdowns
- Writing comments
- Repetitive tasks (data entry)
- Code completions

AI may **NOT** be used in the following ways:
- Full writing of code
- Writing documentation
- Writing comments for work that was not reviewed by a human

Failure to follow these rules may result in your contributions being rejected.
